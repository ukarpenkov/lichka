- Спецификация написана на русском языке для удобства команды
- Структура разделов: обзор → стек → архитектура → слои → UI → навигация → медиа → локализация → паттерны
- Добавлен полный справочник путей к файлам для быстрого навигирования
- Описаны цветовые паттерны (hex-суффиксы для прозрачности) — важный паттерн для UI-багов
