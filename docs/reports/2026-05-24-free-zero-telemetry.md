# Монетизация и telemetry (#75–#76)

**Дата:** 2026-05-24
**Промпт/задача:** Бесплатно, без телеметрии (#75–#76)

## Что сделано
- **#75:** полностью бесплатно, без freemium/IAP/подписок
- **#76:** zero telemetry — без analytics и Crashlytics

## Изменённые файлы
- `docs/spec/white-requirements.md` — #75, #76, changelog

## Принятые решения
- Монетизация не планируется в MVP v1
- Никаких сторонних crash/analytics SDK
- Dev-диагностика — только локальные логи в dev-сборке

## Известные ограничения
- Без crash reporting сложнее отлавливать баги у пользователей — осознанный trade-off

## Тестирование
- Не применимо (только документация)
