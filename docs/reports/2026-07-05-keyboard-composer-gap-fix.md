# Увеличение зазора между блоком ввода и клавиатурой

**Дата:** 2026-07-05
**Промпт/задача:** Исправить баг — блок сообщений при открытой клавиатуре расположен слишком низко, почти касается панели клавиатуры. Увеличить отступ на ~10–12 px.

## Что сделано
- Увеличена константа `KEYBOARD_COMPOSER_GAP` с 4 до 16 px в `src/shared/lib/keyboard.ts`
- Значение 16 = прежние 4 + 12 запрошенных px

## Изменённые файлы
- `src/shared/lib/keyboard.ts:13` — `KEYBOARD_COMPOSER_GAP`: 4 → 16

## Принятые решения
- Зазор задаётся единой константой, которая экспортируется из `shared/lib` и используется в `MessageComposer` через `translateY(-KEYBOARD_COMPOSER_GAP)` — менять достаточно в одном месте

## Известные ограничения
- Визуальный результат зависит от устройства; при необходимости значение можно подкорректировать

## Тестирование
- Ручная проверка: открыть чат → фокус на поле ввода → убедиться, что между полем ввода и клавиатурой виден комфортный отступ
