# DateTimePicker — UX-редизайн

**Дата:** 2026-05-30
**Промпт/задача:** Доработать DateTimePicker: тоньше грани, TimeScroller вписан во внутренний круг, усиление анимации при выборе месяца

## Что сделано
- Уменьшена толщина граней: внешний безель (месяцы) 30→22px, внутренний (даты) 26→18px, зазор 9→7px
- TimeScroller компактнее: ITEM_HEIGHT 34→28, COL_WIDTH 46→38, шрифт выбранного 26→22, соседних 14→11, разделитель ":" 24→20
- Шрифт на безелях 13→11 (соответствует тонким граням)
- Анимация header при выборе месяца: день сдвигается на -12px (было -6) + плавное затемнение opacity до 0.45
- Haptic feedback на каждое изменение дня/месяца — работал и раньше, подтверждён

## Изменённые файлы
- `src/widgets/datetime-picker/geometry.ts` — MONTH_WIDTH 22, DAY_WIDTH 18, RING_GAP 7
- `src/widgets/datetime-picker/TimeScroller.tsx` — ITEM_HEIGHT 28, COL_WIDTH 38, шрифты уменьшены
- `src/widgets/datetime-picker/DateTimePicker.tsx` — fontSize безелей 11, header translateY -12, dayTextStyle opacity анимация

## Принятые решения
- Грани стали тоньше на ~27% — TimeScroller (84px) теперь гарантированно вписывается во внутренний круг (~144px диаметр)
- Затемнение дня при выборе месяца — визуальный сигнал что день сейчас не активен

## Известные ограничения
- Полная проверка возможна только на устройстве/эмуляторе (жесты, haptic, визуальное соответствие)

## Тестирование
- Открыть picker → TimeScroller должен быть внутри внутреннего кольца
- Прокрутить дни → haptic на каждое шаг, число над внешним кольцом
- Прокрутить месяцы → haptic, полное название месяца над кольцом, день сдвигается вверх и затемняется
- Проверить 12/24 формат
