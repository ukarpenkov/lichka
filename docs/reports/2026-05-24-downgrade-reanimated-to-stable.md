# Откат react-native-reanimated на стабильную версию

**Дата:** 2026-05-24
**Промпт/задача:** Вернуть react-native-reanimated и react-native-worklets на стабильные версии вместо ночных

## Что сделано
- Изменена версия `react-native-reanimated` с `^4.4.0-nightly-20260523-7c5580346` на `^4.3.1` (последняя стабильная)
- `react-native-worklets` уже был на стабильной версии `0.8.3` — изменений не потребовалось
- Запущено `npm install` для обновления `package-lock.json`

## Изменённые файлы
- `package.json` — downgrade react-native-reanimated до ^4.3.1
- `package-lock.json` — обновлён автоматически

## Принятые решения
- Ночные версии `4.4.0-nightly-*` заменены на последнюю стабильную `4.3.1`, а не на `4.4.0-rc.*` — стабильные релизы надёжнее для продакшена

## Известные ограничения
- В стабильной `4.3.1` могут отсутствовать фиксы/фичи из ночных сборок `4.4.0`

## Тестирование
- `npm install` завершился успешно (0 vulnerabilities)
