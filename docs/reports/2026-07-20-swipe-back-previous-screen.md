# Свайп назад на предыдущий экран

**Дата:** 2026-07-20
**Промпт/задача:** Свайп назад не должен закрывать приложение — должен возвращать на предыдущую страницу (например ChatRoom → список чатов).

## Что сделано

- На вложенных экранах (`ChatRoom`, `ThemePicker`) отключён горизонтальный свайп корневого `SwipeablePager`, чтобы жест «назад» не перехватывался листанием табов и не уводил с экрана / не конфликтовал с pop стека.
- Для независимых стеков (`NavigationIndependentTree` + `BaseNavigationContainer`) добавлен `BackHandler`: на активном табе системный back / gesture navigation вызывает `goBack()` вложенного стека вместо выхода из приложения (корневой контейнер видит только `Main` и не умеет pop’ать вложенные деревья).
- Включён native-stack edge/fullscreen gesture (`gestureEnabled`, `fullScreenGestureEnabled`) для iOS interactive pop.

## Изменённые файлы

- `src/app/MainTabsContext.tsx` — состояние `tabSwipeEnabled` / `setNestedStackOpen` по индексу таба.
- `src/app/AppNavigator.tsx` — bridge вложенных стеков (state → блок pager, BackHandler), `enabled={tabSwipeEnabled}` у pager.

## Принятые решения

- Блокировать свайп табов только когда **активный** таб находится на вложенном экране (глубина стека > 0). На соседних табах листание по-прежнему работает; переключение через иконки tab bar — всегда.
- BackHandler вешается на уровне каждого независимого контейнера и срабатывает только при совпадении `activeIndex` с табом стека — иначе скрытый `ThemePicker` мог бы съедать back, пока пользователь на чатах.

## Известные ограничения

- Из `ChatRoom` на соседний таб свайпом больше нельзя — сначала назад в список, либо тап по иконке. Это ожидаемо для «свайп назад = предыдущая страница».
- На корневых экранах табов (список чатов / запланировано / настройки) системный back по-прежнему может свернуть/закрыть приложение — истории назад нет.

## Тестирование

- `npx eslint src/app/AppNavigator.tsx` — после удаления неиспользуемого импорта.
- Ручная проверка: ChatList → ChatRoom → свайп/назад → ChatList; Settings → ThemePicker → назад → Settings; на корневых табах свайп между табами сохранён.
