# Fix: обрезка эмодзи по вертикали

**Дата:** 2026-07-01
**Промпт/задача:** Исправить обрезку эмодзи сверху и снизу в пикере

## Что сделано
- Добавлен `lineHeight` в стили эмодзи в `EmojiGrid.tsx` и `ChatForm.tsx`

## Изменённые файлы
- `src/widgets/chat-form/EmojiGrid.tsx:69-73` — добавлен `lineHeight: 52` (равен высоте ячейки) и `textAlign: 'center'`
- `src/widgets/chat-form/ChatForm.tsx:289-292` — добавлен `lineHeight: 96` (равен высоте контейнера аватара)

## Принятые решения
- Значение `lineHeight` установлено равным высоте родительского контейнера — это уже используемый паттерн в `Avatar.tsx:35` (`lineHeight: size`)
- Без `lineHeight` React Native использует значение по умолчанию (~1.2× font-size), которое недостаточно для полного отображения эмодзи на Android — системный рендеринг эмодзи выходит за границы выделенной строки и обрезается
