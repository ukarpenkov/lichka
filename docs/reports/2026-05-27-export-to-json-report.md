# Экспорт в JSON

**Дата:** 2026-05-27
**Промпт/задача:** Реализовать экспорт всех данных приложения в JSON-файл (задача 12.1)

## Что сделано

- Создана функция `exportToJSON()` в `src/features/export/`
- Функция собирает все чаты, сообщения и настройки в JSON с `schema_version: 1`
- Файл сохраняется через `react-native-fs` в `DocumentDirectoryPath/licka-backup-<timestamp>.json`
- Заглушка «Экспорт в файл» в настройках заменена на рабочий вызов с Alert об успехе/ошибке

## Изменённые файлы

- `src/features/export/exportToJSON.ts` — новая функция экспорта
- `src/features/export/index.ts` — barrel export
- `src/features/index.ts` — добавлен реэкспорт `exportToJSON`
- `src/pages/settings/SettingsScreen.tsx` — подключён реальный экспорт вместо заглушки

## Принятые решения

- Timestamps в JSON хранятся в UTC (ISO 8601) — `new Date().toISOString()`
- Сообщения вложены внутрь каждого чата для удобства навигации
- Настройки экспортируются как отдельный объект верхнего уровня
- Имя файла содержит timestamp для уникальности: `licka-backup-2026-05-27T12-00-00.json`
- `react-native-fs` уже установлен — дополнительные зависимости не потребовались

## Известные ограничения

- Медиафайлы (аватары, голосовые) не включены в экспорт — только JSON-данные
- Нет обратного импорта (задача 12.2+)
- Share API не используется — файл сохраняется в DocumentDirectoryPath, путь показывается в Alert

## Тестирование

- TypeScript-компиляция проходит без ошибок
