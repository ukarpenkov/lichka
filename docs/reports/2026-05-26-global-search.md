# Глобальный поиск по сообщениям

**Дата:** 2026-05-26
**Промпт/задача:** 11.1 Реализовать глобальный поиск на экране ChatList

## Что сделано
- Добавлен LIKE-фолбэк в `searchMessages` (FTS5 отключен в op-sqlite)
- `SearchResult` расширен полем `chat_title` для отображения названия чата
- Создан компонент `GlobalSearch` — оверлей поиска на экране ChatList
- В `ChatListScreen` добавлен заголовок с иконкой поиска
- В `ChatRoomScreen` добавлен скролл к сообщению при переходе из результатов поиска

## Изменённые файлы
- `src/shared/db/search.ts` — LIKE-фолбэк, поле `chat_title`, рефакторинг на FTS+LIKE
- `src/pages/chat-list/GlobalSearch.tsx` — новый компонент поиска
- `src/pages/chat-list/ChatListScreen.tsx` — заголовок + иконка поиска + интеграция GlobalSearch
- `src/pages/chat-room/ChatRoomScreen.tsx` — скролл к сообщению по `messageId`

## Принятые решения
- **FTS5 vs LIKE:** FTS5 миграция отключена (op-sqlite без модуля). `searchMessages` пробует FTS5, при ошибке падает в LIKE. Когда FTS5 включат — переключится автоматически
- **Подсветка результатов:** `<mark>` теги парсятся в `HighlightedBody` — совпадения выделяются фоном и жирным шрифтом
- **Навигация:** из результатов поиска тап переходит в `ChatRoom` с `messageId`, чат скроллится к нужному сообщению

## Известные ограничения
- LIKE-поиск без индекса, LIMIT 50 результатов
- При включении FTS5 нужно будет удалить `chat_title` из FTS-запроса (его там не было раньше)

## Тестирование
- TypeScript: изменённые файлы без ошибок
- Визуально: иконка поиска в заголовке, оверлей с полем ввода, результаты с подсветкой, переход в чат
