# Добавление команды /commit для opencode

**Дата:** 2026-07-12
**Промпт/задача:** /commit — автоматический коммит

## Что сделано
- Создана пользовательская команда `/commit` для opencode
- Команда автоматизирует: staging изменений, генерацию commit message по Conventional Commits, коммит и написание отчёта в `docs/reports/`

## Изменённые файлы
- `.opencode/command/commit.md` — новый файл с инструкциями для commit-ассистента

## Принятые решения
- Файл размещён в `.opencode/command/` — стандартное расположение пользовательских команд opencode
- Команда следует правилам AGENTS.md: Conventional Commits, отчёты на русском, без AI-атрибуции

## Известные ограничения
- Хук `.githooks/commit-msg` не установлен как executable — выводится предупреждение при коммите

## Тестирование
- Команда выполнена успешно: `git commit` завершился с hash `2ee817d`
