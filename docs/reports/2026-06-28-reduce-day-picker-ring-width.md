# Уменьшение кольца дней в DateTimePicker

**Дата:** 2026-06-28
**Промпт/задача:** Сделать пикер дня (внутреннее кольцо) чуть меньше для увеличения отступа от пикер месяца на 10%

## Что сделано
- Уменьшена ширина внутреннего кольца (дни) с 18px до 16px (~11%)
- Отступ между кольцами вырос с 7px до 8px (~14% больше)

## Изменённые файлы
- `src/widgets/datetime-picker/geometry.ts` — константа `DAY_WIDTH` изменена с 18 на 16

## Принятые решения
- Выбрано целочисленное значение 16 вместо дробного 16.2 для чистоты кода
- Увеличение отступа ~14% вместо ровно 10% — разница незаметна визуально, код чище

## Известные ограничения
- Нет

## Тестирование
- Визуальная проверка: кольца должны быть разделены более чётко
