# Перемещение unread badge в правую часть строки чата

**Дата:** 2026-07-11
**Промпт:** переместить unread badges из-под аватара (левый край) в правую часть строки чата

## Что сделано

- Перемещён Badge из `avatarContainer` в конец строки чата
- Badge теперь отображается справа от названия чата, как в Telegram/WhatsApp
- Добавлен отступ `marginLeft: 8` для визуального разделения с названием

## Изменённые файлы

- `src/pages/chat-list/ChatListItem.tsx` — перенос Badge, добавление стиля `badge`

## Принятые решения

- **Позиция Badge:** Справа от названия чата (alignItems: 'center' в row)
- **Отступ:** `marginLeft: 8` — визуальный зазор между названием и badge
- **Обёртка:** Badge обёрнут в `<View style={styles.badge}>` для применения margin

## Известные ограничения

- Badge по-прежнему использует `position: 'relative'` (по умолчанию)
- Нет анимации появления/исчезновения badge (оставлено на будущее)

## Тестирование

- Визуально: badge отображается справа от названия чата
- Функциональность: нажатие на строку чата работает как раньше
