# Иконки типа сообщения в чате

**Дата:** 2026-07-03
**Промпт/задача:** Добавить маленькую иконку с типом сообщения рядом с временем отправки в чате (кроме голосовых). Размер иконки — как размер времени отправки. Иконки — как в блоке отправки сообщений.

## Что сделано
- В `MessageBubble.tsx` добавлена строка с иконкой типа сообщения в `metaRow` перед временем отправки
- Иконка показывается только для типов `reminder` (Bell), `alarm` (AlarmClockIcon), `periodic` (Repeat)
- Для `simple` (текст) и голосовых сообщений иконка не показывается
- Размер иконки (11px) совпадает с fontSize времени отправки
- Создан feature proposal в `docs/features/chat-message-type-icons.md`

## Изменённые файлы
- `src/pages/chat-room/MessageBubble.tsx` — добавлены импорты иконок, TYPE_ICON map, отрисовка иконки в metaRow
- `docs/features/chat-message-type-icons.md` — создан proposal

## Принятые решения
- `simple` сообщения не получают иконку — это тип по умолчанию, иконка была бы избыточной
- Голосовые сообщения исключены явно (как указано в задаче)
- Паттерн `TYPE_ICON` повторяет существующий в `ScheduledItem.tsx`
- Иконка `alarm` обрабатывается отдельно (как и в ScheduledItem), т.к. AlarmClockIcon — кастомный SVG, а не lucide-компонент
