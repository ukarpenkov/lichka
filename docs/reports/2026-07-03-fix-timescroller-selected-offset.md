# TimeScroller — фикс смещения индикации выбранного элемента

**Дата:** 2026-07-03
**Промпт/задача:** Исправить баг: в пикере времени голубая подсветка применяется к верхнему элементу вместо центрального

## Что сделано
- Исправлено смещение `scrollToOffset` — убран `+ PAD` (строка 50), теперь скролл идёт на `index * ITEM_HEIGHT`
- Исправлены все 4 обработчика скролла — убрано `- PAD` при вычислении `idx` (строки 57, 69, 81, 93)
- Теперь выбранный элемент (центр списка из 3 элементов) получает accent-цвет, а верхний и нижний — нейтральный

## Изменённые файлы
- `src/widgets/datetime-picker/TimeScroller.tsx` — 5 строк исправлено

## Принятые решения
- Причина бага: `scrollToOffset((index + PAD) * ITEM_HEIGHT)` скроллил элемент на верхнюю позицию, а highlight-ячейка находится на центральной (`top: ITEM_HEIGHT * 1`). Акцентный цвет применялся к элементу вверху, а не по центру.
- Исправление минимальное: убрать `PAD` из обоих мест (scrollToOffset и обработчики), сохранив `PAD` только для пустых спейсеров в данных.
