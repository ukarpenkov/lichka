# Редизайн экрана будильника

**Дата:** 2026-07-05
**Промпт/задача:** Переделать дизайн будильника по прототипу `design/layout/alarm2.0.html`, используя цветовую схему приложения

## Что сделано
- Полный редизайн `AlarmScreen` в соответствии с прототипом
- Добавлена верхняя панель с заголовком «Будильник» и текущим временем
- Обновлены размеры шрифтов: время 64px (weight 200), лейбл 20px (weight 500)
- Обновлены анимации: пульс (2400ms, scale 1→1.5), тряска (3deg, 600ms)
- Кнопка «Отключить» с иконкой X, кнопка «Отложить» с иконкой часов
- Добавлен home indicator внизу
- Убран внутренний ring (оставлен один)
- Все цвета выводятся из темы (text + alpha)

## Изменённые файлы
- `src/pages/alarm/AlarmScreen.tsx` — полный редизайн

## Принятые решения
- Размер иконки-обёртки фиксированный (128px), как в прототипе
- Деструктивный цвет `#FF6B6B` оставлен константой, т.к. это семантический цвет
- Цвета всех остальных элементов — производные от `text` с альфа-суффиксами (`text + '66'`, `text + '14'` и т.д.)
- Иконка X из `lucide-react-native` (уже используется в проекте)
