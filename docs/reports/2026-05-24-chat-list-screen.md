# Экран списка чатов (ChatList)

**Дата:** 2026-05-24
**Промпт/задача:** Реализация экрана ChatList (задача 5.1 из tasks)

## Что сделано

- Создан компонент `Avatar` в `shared/ui` — круглый аватар 48dp с fallback на первую букву названия
- Создан `ChatListItem` — элемент списка с аватаром и названием чата, поддержка tap и long-press
- Создан `ChatContextMenu` — модальное контекстное меню с пунктами «Редактировать» и «Удалить» (красный)
- Переписан `ChatListScreen`:
  - Загрузка чатов из `getChats()` при фокусе экрана (`useFocusEffect`)
  - `FlatList` со списком чатов
  - Тап → навигация в `ChatRoom` с `chatId`
  - Долгое нажатие → контекстное меню
  - Удаление с подтверждением через `Alert.alert`
  - FAB «+» справа снизу (заглушка — пока без действия)
  - Пустое состояние: центрированный текст «Создайте первый чат»

## Изменённые файлы

- `src/shared/ui/Avatar.tsx` — новый компонент аватара
- `src/shared/ui/index.ts` — добавлен экспорт Avatar
- `src/pages/chat-list/ChatListScreen.tsx` — полная реализация экрана
- `src/pages/chat-list/ChatListItem.tsx` — элемент списка чатов

## Принятые решения

- FAB «+» — заглушка (нет экрана создания чата)
- «Редактировать» — заглушка (TODO)
- Аватар: fallback с полупрозрачным фоном `text + '15'`
- Контекстное меню через RN `Modal`, а не через библиотеку
- Иконки через `react-native-svg` (консистентно с навигатором)

## Известные ограничения

- FAB не создаёт чат (нет UI для ввода названия)
- «Редактировать» не реализовано (нет экрана редактирования)

## Тестирование

- Визуально: пустой экран → текст «Создайте первый чат»
- После добавления чата → отображается в списке с аватаром-буквой
- Тап → навигация в ChatRoom
- Долгое нажатие → модалка → Удалить → подтверждение → чат удаляется
