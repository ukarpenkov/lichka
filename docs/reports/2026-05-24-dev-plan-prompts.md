## Принятые решения

- Промты написаны на русском языке (consistent с ТЗ)
- Каждый промт — атомарная задача, понятная LLM
- Правило: перед следующим промтом проверять что предыдущий `[x]`
- Итоговый checkpoint: «Всё хорошо» после всех промтов
- План учитывает FSD-архитектуру и порядок зависимостей (shared → entities → features → pages → app)

## Известные ограничения

- Google Drive интеграция (промт 12.3) требует OAuth setup — может потребовать ручной настройки в Google Cloud Console
- Нативные модули (промты 8.x, 9.x) требуют работы с Kotlin — LLM может потребоваться контекст android-проекта
- Shared Element Transitions — experimental в Reanimated 4, может не работать на всех устройствах
