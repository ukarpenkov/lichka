# Проверка: кастомный диалог (вместо системного Alert)

- [ ] **Удалить сообщение** — вместо серого системного окна появляется **тёмный попап** в стиле приложения
- [ ] **Удалить чат** — то же самое, диалог стилизован
- [ ] **Тёмная тема** — фон диалога тёмный, текст белый, кнопки аккуратные
- [ ] **Светлая тема** — переключить тему, диалог светлый
- [ ] **Анимация** — окно плавно появляется (fade + zoom)
- [ ] **Кнопка Cancel** — закрывает диалог без действия
- [ ] **Кнопка Delete** — выполняет удаление, диалог закрывается
- [ ] **Тап вне окна** — диалог закрывается
- [ ] **Настройки → экспорт/импорт** — все сообщения об успехе/ошибке тоже в кастомном диалоге
