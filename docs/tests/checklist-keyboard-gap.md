# Проверка: зазор между клавиатурой и полем ввода

- [ ] **Чат открывается** — поле ввода видно внизу
- [ ] **Тапнуть в поле ввода** — клавиатура появляется, поле ввода поднимается сразу с ней (синхронно)
- [ ] **Нет чёрного зазора** — между верхним краем клавиатуры и блоком ввода **нет** пустого пространства
- [ ] **Свернуть клавиатуру** (кнопка Back на Android) — поле ввода опускается обратно, зазора нет
