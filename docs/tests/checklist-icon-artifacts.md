# Проверка: артефакты иконок (яркие точки на стыках линий)

- [ ] **Нижняя панель: иконка «Чаты»** — баббл без жирной точки в месте соединения хвостика с кругом
- [ ] **Нижняя панель: иконка «Календарь»** — линии ровные, без утолщений на стыках
- [ ] **Нижняя панель: иконка «Настройки»** — шестерёнка без жирных пятен в зубцах
- [ ] **Панель действий чата: иконка «Будильник»** — ножки будильника без точек в месте пересечения с корпусом
- [ ] **Панель действий чата: иконка «Микрофон»** — стык ножки и головки без утолщения
- [ ] **Другие иконки в приложении** (колокольчик, повторить, поиск, крестик) — линии везде ровные, без локальных утолщений

> **Как смотреть:** открыть тёмную тему, иконки хорошо видны. Если точки/утолщения есть — они заметны сразу.
