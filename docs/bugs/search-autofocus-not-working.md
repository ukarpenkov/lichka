# Баг: Клавиатура не открывается при активации поиска в списке чатов

## Описание
При нажатии на иконку поиска на экране списка чатов поле ввода появляется, но клавиатура не открывается автоматически. Требуется лишний тап по инпуту, чтобы начать печатать.

## Причина
`inputRef.current?.focus()` вызывался синхронно в `useEffect`, но `GlobalSearch` обёрнут в `Animated.View` с `SlideInDown` (200ms + spring). На момент вызова `focus()` компонент ещё не завершил анимацию входа, и на iOS клавиатура не появляется.

## Решение
Вызов `focus()` обёрнут в `InteractionManager.runAfterInteractions` — дожидается завершения всех анимаций и нативных переходов перед фокусом.
