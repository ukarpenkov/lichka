# Визуальный баг: белая вспышка на правом краю при навигации

**Дата:** 2026-07-03

## Описание

При переходе между списком чатов и окном переписки (NativeStack slide-from-right анимация) на правом краю экрана появляется яркая белая вертикальная вспышка. Баг симметричен — проявляется как при открытии чата, так и при возврате назад.

## Шаги воспроизведения

1. Открыть приложение (тёмная тема, `#000000` фон)
2. Нажать на любой чат в списке
3. Наблюдать правый край экрана во время анимации перехода — белая полоса
4. Нажать «Назад» — вспышка повторяется

## Ожидаемый результат

Плавная анимация без артефактов; область перехода должна быть чёрной.

## Фактический результат

Яркая белая вертикальная полоса на правом краю экрана на момент анимации.

## Окружение

- **OS:** Android (тёмная тема)
- **RN:** 0.85.3
- **Навигация:** `@react-navigation/native-stack` v7.15.1, `react-native-screens` v4.25.2

## Причина

1. **`android:windowBackground`** — тема `Theme.AppCompat.DayNight.NoActionBar` наследует белый фон (`#FFFFFF`). Во время native stack-анимации, пока JS не отрисовал тёмный фон, через правый край «выглядывает» белый background окна.
2. **Нет `contentStyle`** — `Stack.Navigator` не задаёт `contentStyle: { backgroundColor }`, поэтому native Screen-контейнер использует дефолтный белый фон.

## Решение

- Переопределить `android:windowBackground` в `styles.xml`
- Установить `contentStyle` с цветом фона темы в `Stack.Navigator`
- Добавить `StatusBar` с правильным цветом фона
