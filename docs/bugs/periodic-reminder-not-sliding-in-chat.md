# Периодические напоминания не создают «скользящее» сообщение в чате

**Дата обнаружения:** 2026-07-05
**Тип сообщения:** periodic
**Статус:** fixed

## Симптом
Периодическое напоминание не ведёт себя как одно «живое» сообщение:
- Сообщение всегда отображается на позиции создания (не «скользит» вперёд)
- Нет механизма удаления старой инстанции и создания новой при каждом срабатывании
- В чате всегда видно само «тело» шаблона, а не свежий экземпляр

## Шаги воспроизведения
1. Создать сообщение в `12:00` с типом **периодическое** и интервалом `5 минут`
2. Убедиться, что в Scheduled — 1 запись
3. Дождаться срабатываний

## Причина
1. `getVisibleMessagesByChatId()` включает периодические сообщения через `type IN ('simple', 'periodic', 'image')` — шаблон всегда виден в чате на своей оригинальной позиции
2. Нет кода, который при срабатывании создавал бы новый экземпляр сообщения и удалял старый
3. Нативная сторона (`AlarmReceiver.kt`) только показывает уведомления и перепланирует alarm — не трогает SQLite

## Исправление
1. **`messageRepository.ts`**:
   - `getVisibleMessagesByChatId()` — исключить `periodic` из чата (шаблон больше не отображается как обычное сообщение)
   - Добавить `getPeriodicDisplayMessages(chatId)` — генерирует синтетические сообщения для «сработавших» периодических:
     * Вычисляет время последнего срабатывания: `createdAt + floor((now - createdAt) / interval) * interval`
     * Возвращает **ровно одно** синтетическое сообщение на каждый шаблон (самое свежее)
     * ID: `periodic:<templateId>` — уникальный, не меняется между срабатываниями
2. **`ChatRoomScreen.tsx`**:
   - `loadData()` объединяет обычные сообщения + периодические display-сообщения
   - Добавлен периодический рефреш (каждые 30 с) для пересчёта позиции
   - Обработчики delete/edit адаптированы для синтетических ID (извлекают templateId)

## Изменённые файлы
- `src/entities/message/model/messageRepository.ts` — `getPeriodicDisplayMessages()`, исключение periodic из `getVisibleMessagesByChatId`
- `src/entities/message/index.ts` — экспорт `getPeriodicDisplayMessages`
- `src/pages/chat-room/ChatRoomScreen.tsx` — интеграция display-сообщений, polling, адаптация delete/edit
