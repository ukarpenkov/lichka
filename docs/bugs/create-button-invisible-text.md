# Баг: Текст на кнопке Create не читается

## Описание
Кнопка "Создать" / "Create" в форме чата была полностью чёрной — текст на ней не был виден на любом фоне темы.

## Причина
`Button.tsx` использовал `color: text` (цвет текста темы) для надписи, а `ChatForm` передавал `backgroundColor: text` (тот же цвет темы) как фон кнопки. В результате фон и текст имели одинаковый цвет.

## Решение
`Button.tsx` теперь определяет `backgroundColor` из переданных стилей и инвертирует цвет текста: если фон совпадает с `text` темы, текст становится `background`. Автоматически работает на всех 13 темах.
